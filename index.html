━━ 開始之前：確認環境 ━━
  • 安裝 VS Code（免費）— code.visualstudio.com
  • 準備 Claude Pro 帳號（約 NT$650／月）或 Anthropic API Key
    ⚠ 免費帳號無法使用 Claude Code，登入後會看到升級提示
  • 在 VS Code 安裝 Claude Code Extension（按 Cmd+Shift+X，搜尋「Claude Code」，選 Anthropic 出品那個）
環境不確定有沒有裝好？先從教材一第 1 章走一遍，確認登入成功再繼續。
━━ 怎麼使用這份教材 ━━
1
用 VS Code 打開這個資料夾：File → Open Folder，選到 claude-code-quickstart/
2
從左側欄找到 quickstart/01-install/README.md，右鍵 → Open Preview（或按 Cmd+Shift+V）開啟排版好的預覽 看到一堆 # 和 ** 符號代表還沒開預覽，用上面的方法切換一下
3
按章節順序走，每個資料夾裡的 README.md 就是該章的教學
4
進入進階工作坊時，切換資料夾：File → Open Folder，改選 workshop-vscode-claude/ 必須切換資料夾，Skills 的 / 指令才能正確出現

━━ 為什麼值得投資這些設定 ━━

CLAUDE.md、Skills 這些設定，不管你用桌面版 Code 模式還是 VS Code，都放在同一個專案資料夾裡、兩邊都能讀。你不需要選邊站。

更值得記住的是一個概念：Portable（可攜帶性）。

你的 CLAUDE.md 是文字檔，Skills 是文字檔，你累積的知識庫也是文字——這些東西存在你自己的電腦裡，不鎖在任何平台。如果哪天出現更強的 AI，把同一批文件接上新工具，一切照舊。

把心力投資在建立你的 CLAUDE.md 和 Skills，才是真正值得的事。AI 是工具，你的文件和知識才是資產。
