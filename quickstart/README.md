# Claude Code 快速入門教學

歡迎！這份教學會帶你從零開始，學會在 VS Code 裡使用 Claude Code。

不需要任何程式基礎，跟著步驟走就好。

---

## 教學地圖

| 章節 | 內容 | 預估時間 |
|------|------|----------|
| [01 - 安裝設定](./01-install/README.md) | 裝好環境、登入帳號 | 10 分鐘 |
| [02 - 第一次對話](./02-hello-claude/README.md) | 開口問 Claude 問題 | 10 分鐘 |
| [03 - 幫我修程式](./03-fix-my-code/README.md) | 讓 Claude 找 bug、解釋程式碼 | 15 分鐘 |
| [04 - 幫我做東西](./04-build-something/README.md) | 給 Claude 一個任務，看它完成 | 15 分鐘 |
| [05 - 實用技巧](./05-tips/README.md) | 讓 Claude 更好用的小撇步 | 10 分鐘 |

**總計約 60 分鐘。**

---

## 使用方式

1. 用 VS Code 打開這個資料夾
2. 從 `01-install` 開始，按順序讀每個資料夾裡的 `README.md`
3. 遇到練習題就動手試試看，不用怕做錯

> 卡住的時候，直接問 Claude 就好 — 這本來就是它存在的意義。
