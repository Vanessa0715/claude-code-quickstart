# Prompt 範本集

以下是幾個可以直接複製貼到 Claude 的任務，挑一個試試看。

做出來的是一個 HTML 檔案，存檔後用瀏覽器打開就能用。推到 GitHub 並開啟 GitHub Pages 後，會有一個固定網址，手機也可以直接瀏覽。

## 運營工具類

**廠商比價表**
```
幫我做一個單一 HTML 檔案的 webapp，功能是廠商比價記錄：
- 可以輸入廠商名稱、項目、報價金額
- 新增後顯示成一個表格
- 自動標出最低價
- 版面要手機也能看清楚
不需要後端，資料用 localStorage 儲存，重新整理後不消失。
```

**費用報銷清單**
```
幫我做一個單一 HTML 檔案的 webapp，功能是費用報銷記錄：
- 可以輸入花費項目、金額、日期、類別（交通／餐費／物料／其他）
- 顯示成清單，底部自動加總
- 可以按類別篩選
- 版面要手機也能操作
不需要後端，資料用 localStorage 儲存，重新整理後不消失。
```

## 升級：跨裝置同步版

上面的工具預設用 localStorage，資料只存在當下這台裝置的瀏覽器裡。
如果需要**多台裝置共用**或**團隊協作**，可以升級成 Firebase 版。

**事前準備（一次性設定）：**
1. 前往 [Firebase Console](https://console.firebase.google.com/)，用 Google 帳號登入
2. 建立新專案，開啟 Firestore Database（選 test mode 即可開始）
3. 在專案設定裡找到「你的應用程式」，複製那段設定碼（firebaseConfig）

**升級任何一個 prompt 的方法：**

把這段補充加在原本 prompt 的結尾：

```
資料不要用 localStorage，改成串接 Firebase Firestore 來儲存。
以下是我的 Firebase 設定碼，請直接寫進 HTML 裡：

[貼上你的 firebaseConfig]

這樣不同裝置打開同一個網址，資料會即時同步。
```
