# 04 — 幫我做東西

這一章是最有趣的部分：**從零開始，讓 Claude 幫你做一個東西**。

你不需要知道怎麼寫，只要描述清楚你想要什麼。

---

## 怎麼給 Claude 任務？

好的 prompt 通常包含三件事：

1. **你要做什麼**（功能描述）
2. **限制條件**（語言、格式、簡單或複雜）
3. **預期的結果**（要輸出什麼）

範例：
```
幫我做一個單一 HTML 檔案的 webapp，功能是廠商比價記錄。
可以輸入廠商名稱和報價，新增後顯示成表格，並自動標出最低價。
版面要手機也能看清楚，不需要後端。
```

---

## 練習：挑一個任務試試看

打開 `prompt-examples.md`，裡面有幾個可以直接使用的任務範本。

挑一個你覺得有趣的，貼到 Claude 對話框，看它做出什麼。

---

## 如果結果不對怎麼辦？

直接繼續說就好，Claude 會根據你的回饋調整：

```
這個方向對，但我希望輸出的格式改成清單，不要用表格。
```

```
這樣太複雜了，幫我簡化成三個步驟就好。
```

```
加上錯誤提示，如果使用者輸入空白要顯示警告。
```

這種來回調整的過程叫做「迭代」，是使用 Claude 最重要的技巧。
