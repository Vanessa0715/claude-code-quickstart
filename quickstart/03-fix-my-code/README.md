# 03 — 幫我修程式

這一章要讓 Claude 幫你看一段**故意寫錯的程式碼**，找出問題並修正。

不需要看懂程式碼，這正是重點：你不用懂，Claude 懂就好。

---

## 準備

打開這個資料夾裡的 `broken.py` 檔案，先滾動看一下，大概知道它在做什麼。

（它是一個計算行銷活動費用的小程式，裡面藏了幾個錯誤。）

---

## 練習一：讓 Claude 解釋這段程式碼

1. 打開 `broken.py`，全選（`Cmd+A`）
2. 按右鍵 → Ask Claude，或複製後貼到對話框
3. 輸入：

```
這段程式碼在做什麼？用簡單的中文說明，不要用專業術語。
```

---

## 練習二：讓 Claude 找 bug

繼續在對話框輸入：

```
這段程式碼有沒有錯誤？幫我找出來並說明問題在哪裡。
```

看看 Claude 能不能找到所有的問題。

---

## 練習三：讓 Claude 直接幫你改檔案

這是 Claude Code 和一般 ChatGPT 最不一樣的地方：**它可以直接修改你電腦上的檔案**，不需要你自己複製貼上。

1. 確認 `broken.py` 已經在 VS Code 裡打開
2. 在 Claude 對話框輸入：

```
@broken.py 這段程式碼有幾個錯誤，幫我直接修好它。
```

3. Claude 會列出發現的問題，然後在畫面上顯示它想做的修改（會用綠色/紅色標出差異）
4. 確認修改內容後，點「**Accept**」讓它直接套用到檔案

修改套用後，打開 `broken.py`，就會看到已經是改好的版本了。

> **這就是「代理模式」：** Claude 不是給你答案讓你自己貼，而是直接動手做。這個模式適合任何「幫我改、幫我建、幫我刪」的任務。

---

## 小觀察

注意 Claude 修改前會：
- 先說明它找到幾個問題
- 顯示「diff 預覽」讓你看清楚哪裡被改了
- 等你確認才執行

這個確認步驟很重要——你隨時可以點「Reject」拒絕某個修改。
