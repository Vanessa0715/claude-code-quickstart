# 02 — 第一次對話

Claude 可以回答問題、整理文字、解釋東西。這一章先從最簡單的開始：**讓它幫你整理一段亂掉的文字**。

---

## 認識介面

打開 VS Code 左側的 Claude 面板，你會看到一個對話框。

- 直接在底部輸入框打字，按 Enter 送出
- Claude 會在上方顯示回覆
- 按 `+` 可以開新對話（清空記憶）

---

## 練習一：直接問問題

在對話框裡輸入任何你好奇的問題，例如：

```
什麼是程式語言？用一句話解釋給完全沒學過的人聽。
```

送出後看看 Claude 怎麼回答。

---

## 練習二：整理文字

這個資料夾裡有一個 `sample.txt` 檔案，裡面是一份**故意弄亂的工作清單**。

**做法：**
1. 打開 `sample.txt`
2. 全選文字（`Cmd+A` / `Ctrl+A`）
3. 複製（`Cmd+C` / `Ctrl+C`）
4. 到 Claude 對話框，輸入：

```
幫我把以下這份清單整理成有條理的格式，依照類別分組：

[貼上你複製的文字]
```

5. 看看 Claude 整理出來的結果

---

## 練習三：用 @ 直接把檔案丟給 Claude

不用手動複製，Claude Code 有更快的方式：

1. 在 Claude 對話框裡輸入 `@`
2. 會跳出選單，找到 `sample.txt` 並點選
3. 補上你的問題，例如：

```
@sample.txt 這個清單裡哪些事情有截止時間？幫我整理出來。
```

`@` 可以引用檔案、資料夾，甚至整個專案，是 Claude Code 裡最常用的技巧之一。

---

## 兩種使用模式

到目前為止你用的都是「**對話模式**」——你問，Claude 回答，文字只在聊天視窗裡。

Claude Code 還有另一種更強大的模式：「**代理模式**」——Claude 可以直接幫你讀寫電腦上的檔案、執行指令，不用你自己複製貼上。
