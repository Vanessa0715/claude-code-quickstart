# Claude Code 進階工作坊

## 你今天會學到什麼

Claude Code 不只是一個「對話框」。它有四個工具可以讓它變成真正懂你的工作助手：

| 工具 | 一句話說明 |
|------|-----------|
| **CLAUDE.md** | 告訴 Claude 你是誰、你怎麼工作 |
| **Skills** | 存下你常用的指令，一個 `/` 就觸發 |
| **MCP** | 讓 Claude 直接操作 Notion、Google Calendar |
| **Plugin** | 別人打包好的工具包，一鍵安裝 |

---

| 章節 | 內容 | 預估時間 |

## 開始之前

確認你已經完成：
- [ ] 安裝好 VS Code
- [ ] 安裝並登入 Claude Code extension
- [ ] **用 VS Code 單獨打開 `workshop-vscode-claude` 這個資料夾**（重要）

> **為什麼要單獨開這個資料夾？**
> Skills 的觸發範圍以「VS Code 目前開啟的資料夾」為準。
> 如果你是打開上層的 `claude-code-quickstart`，裡面的 Skills 就不會出現在 `/` 選單。
>
> **做法：** 在 VS Code 上方選單點 `File → Open Folder`，選到 `workshop-vscode-claude` 這一層，確認左側標題列顯示的是 `WORKSHOP-VSCODE-CLAUDE`。

都準備好了？從 [01 - CLAUDE.md](./01-claude-md/README.md) 開始。
