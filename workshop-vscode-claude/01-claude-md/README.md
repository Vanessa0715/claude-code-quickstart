# 01 — CLAUDE.md：告訴 Claude 你是誰

## 你會學到什麼

建立一個 `CLAUDE.md` 檔案，讓 Claude 在每次開啟這個專案時，都自動知道你的背景、工作方式和偏好。

**為什麼這很重要？**
> 沒有 CLAUDE.md：每次對話 Claude 都把你當陌生人，什麼都要重新解釋。
>
> 有了 CLAUDE.md：Claude 一開口就知道你是行銷主管、習慣用繁體中文、不喜歡技術術語。

---

## CLAUDE.md 是什麼？

它就是一個純文字檔案，名字就叫 `CLAUDE.md`，放在你的專案資料夾裡。

Claude Code 在你每次打開對話時，會**自動讀取這個檔案**，把裡面的設定當作對話的前提。

你可以在裡面寫：
- 你的職位和工作內容
- 你希望 Claude 用什麼語氣回答
- 你的團隊常用的術語或慣例
- Claude 絕對不能做的事

---

## 步驟

### Step 1：打開模板

點開這個資料夾裡的 `my-claude-md/CLAUDE.md`，這是一份已經幫你準備好框架的模板。

### Step 2：填寫你的資料

把模板裡用 `[ ]` 標示的部分換成你自己的資訊。

不用寫得很完美，先有內容比較重要。

### Step 3：存檔

按 `Cmd+S`（Mac）/ `Ctrl+S`（Windows）存檔。

### Step 4：測試

打開 Claude Code 對話框，輸入：

```
你好，根據我的 CLAUDE.md，你知道我是做什麼的嗎？請簡單介紹一下你對我的理解。
```

如果 Claude 能正確描述你的背景，代表它已經讀到你的設定了。

---

## 常見問題

**Q：CLAUDE.md 放哪裡？**
放在你「用 VS Code 打開的那個資料夾」的根目錄就好。比如你打開的是 `my-project/`，就放在 `my-project/CLAUDE.md`。

**Q：可以有多個 CLAUDE.md 嗎？**
可以。不同專案有不同的 CLAUDE.md，Claude 會讀當前打開的那個。你也可以在家目錄放一個全域的。

**Q：裡面要用什麼語言寫？**
隨便，中文英文都可以。Claude 都看得懂。
